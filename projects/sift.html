SIFT

SIFT - Feedback Summarization

The goal of SIFT is to take large amounts of textual data to provide useful information including general sentiment, text summarization and topic modeling.

Technologies: Javascript (React + Redux + JEST), Python (NLTK + Pytest), Go, Celery, Redis, RabbitMQ, Docker, Travis CI, ESLint


SIFT contains three repositories. Project is run with docker that opens up an instance of each repository in a container
